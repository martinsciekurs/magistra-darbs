# graphrag/callbacks/llm_callbacks.py

## Overview

Base callback interface for handling LLM token generation events.

This module defines a Protocol-based base class for callbacks that respond to token generation events in an LLM pipeline. The primary export is the BaseLLMCallback protocol, which defines on_llm_new_token(token) for subclasses to override and implement custom behavior as new tokens are produced.

Key exports:
- BaseLLMCallback

Public API:
on_llm_new_token(token: str) - Handle when a new token is generated.
    Args:
      token: The new token generated by the LLM.
    Returns:
      None
    Raises:
      None
