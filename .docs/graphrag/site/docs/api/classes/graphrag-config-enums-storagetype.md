# StorageType

**File:** `graphrag/config/enums.py`

## Overview

Enumeration of storage and configuration key types used by graphrag.

Represents the various storage backends and related configuration keys that the project may utilize. Each member corresponds to a string key used in the configuration system.

Note: The __repr__ method returns the member's value wrapped in double quotes.

Args:
    None: The StorageType Enum does not accept initialization parameters; members are defined on the class.

Returns:
    StorageType: The enumeration type itself; members represent specific storage/config keys.

Raises:
    None: This class does not raise exceptions during normal usage.
