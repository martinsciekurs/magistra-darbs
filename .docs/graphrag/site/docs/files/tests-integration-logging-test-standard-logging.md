# tests/integration/logging/test_standard_logging.py

## Overview

Integration tests for Graphrag's standard logging configuration.

This module contains integration tests validating the standard_logging module's behavior, including logger naming, logger hierarchy propagation, and various init_loggers configurations (file-based, verbose mode, and custom filename).
