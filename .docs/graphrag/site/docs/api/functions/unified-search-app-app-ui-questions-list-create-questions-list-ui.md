## Signature

def create_questions_list_ui(sv: SessionVariables)

## Description

Create and render a Streamlit UI component to display a list of generated questions and update the selected question when a row is selected.

Args:
    sv (SessionVariables): The session state object containing generated_questions and selected_question attributes used to render the UI and handle selection updates.

Returns:
    None: This function does not return a value.
